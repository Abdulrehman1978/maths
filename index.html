Numerical Methods – Newton-Raphson Method

Assignment 03 | Mini Project

1. Introduction

Numerical Methods are mathematical techniques used to solve problems that cannot be solved analytically or are too complex for exact solutions. These methods provide approximate solutions through iterative algorithms and are widely used in engineering, physics, computer science, and applied mathematics.

Root-finding methods are a crucial category of numerical methods used to find solutions to equations of the form f(x) = 0. These methods are essential in various applications such as optimization problems, solving differential equations, and modeling real-world phenomena. Among the various root-finding techniques, the Newton-Raphson method stands out for its rapid convergence and efficiency.

2. Definition

The Newton-Raphson Method (also known as Newton's Method) is an iterative numerical technique for finding successively better approximations to the roots (or zeros) of a real-valued function. It uses the tangent line at a current approximation to find the next approximation.

The method starts with an initial guess x₀ and generates a sequence of approximations x₁, x₂, x₃, ... that converge to the actual root of the equation f(x) = 0, provided the initial guess is sufficiently close to the root and the function satisfies certain conditions.

3. Algorithm

The step-by-step algorithm for the Newton-Raphson Method is as follows:

  1. Start: Begin the process.
  2. Define the function: Input the function f(x) whose root is to be found.
  3. Define the derivative: Input the derivative f'(x) of the function.
  4. Input initial guess: Choose an initial approximation x₀ close to the actual root.
  5. Input tolerance: Set the desired accuracy (tolerance) ε and maximum number of iterations.
  6. Initialize iteration counter: Set n = 0.
  7. Calculate function values: Compute f(xₙ) and f'(xₙ).
  8. Check derivative: If f'(xₙ) = 0, the method fails (division by zero). Stop and choose a different initial guess.
  9. Apply Newton-Raphson formula: Calculate the next approximation: xn+1 = xn - f(xn) / f'(xn).
  10. Check convergence: If |xn+1 - xn| < ε or |f(xn+1)| < ε, then xn+1 is the required root. Go to step 12.
  11. Update and repeat: Set n = n + 1, xn = xn+1, and go back to step 7. Check if maximum iterations exceeded.
  12. Output result: Display the root xn+1 and the number of iterations.
  13. Stop: End the process.

4. Mathematical Formula

The Newton-Raphson Iterative Formula:

xn+1 = xn - f(xn) / f'(xn)

Where:

  • xn = current approximation
  • xn+1 = next (improved) approximation
  • f(xn) = value of the function at xn
  • f'(xn) = value of the derivative at xn

6. Example

Problem:

Find the root of the equation f(x) = x³ - x - 1 = 0 using the Newton-Raphson method with initial guess x₀ = 1.5 and tolerance ε = 0.0001.

Solution:

Given:

  • f(x) = x³ - x - 1
  • f'(x) = 3x² - 1
  • x₀ = 1.5
  • ε = 0.0001

Iterations:

Iteration (n) xₙ f(xₙ) f'(xₙ) xn+1 |xn+1 - xₙ|
0 1.5000 0.8750 5.7500 1.3478 0.1522
1 1.3478 0.1006 4.4460 1.3252 0.0226
2 1.3252 0.0020 4.2684 1.3247 0.0005
3 1.3247 0.0000 4.2669 1.3247 0.0000

Result: The root of the equation is approximately x ≈ 1.3247 after 3 iterations.

7. Advantages and Disadvantages

✓ Advantages

  • Rapid Convergence: The method exhibits quadratic convergence near the root, making it much faster than linear methods like the Bisection method.
  • High Accuracy: Provides highly accurate results in fewer iterations when the initial guess is close to the actual root.
  • Simple Formula: The iterative formula is straightforward and easy to implement in programming.
  • Wide Applicability: Can be used for a variety of functions including polynomial, transcendental, and algebraic equations.
  • Efficient: Requires fewer function evaluations compared to many other root-finding methods.

✗ Disadvantages

  • Requires Derivative: The method requires the calculation of the derivative f'(x), which may be difficult or impossible for some functions.
  • Sensitive to Initial Guess: Poor choice of initial approximation may lead to divergence or convergence to a different root.
  • Division by Zero: The method fails if f'(x) = 0 at any iteration point.
  • May Not Converge: For functions with multiple roots, inflection points, or discontinuities, the method may oscillate or diverge.
  • No Interval Guarantee: Unlike the Bisection method, it doesn't guarantee finding a root within a specific interval.
  • Computational Cost: Calculating derivatives can be computationally expensive for complex functions.
